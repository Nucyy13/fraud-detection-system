# 💳 Credit Card Fraud Detection System

## 📌 Overview
This project is a web-based application developed to detect fraudulent credit card transactions by analyzing historical transaction data. It helps identify suspicious patterns using rule-based logic and ensures secure handling of user data.

---

## 🛠️ Tech Stack
- **Backend:** Python, Django  
- **Frontend:** HTML, CSS, Bootstrap  
- **Database:** MySQL  

---

## ✨ Features
- 🔍 Detects suspicious transactions using rule-based logic  
- 🔐 Secure user authentication and authorization system  
- 📊 Analysis of historical transaction data  
- 💻 Responsive user interface  
- 🛡️ Secure data handling and validation  

---

## ⚙️ How It Works
1. User logs into the system securely  
2. Transaction data is processed from the database  
3. Rule-based logic is applied to identify suspicious patterns  
4. Flagged transactions are displayed to the user
